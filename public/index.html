All of NarcoNations. Zero cloud. Fully grounded.

mcp-nn is the local research engine that keeps every PDF, dossier, export, and transcript ready for instant recall. Embed and search completely offline, enforce precise citations, and let any MCP-enabled GPT retrieve the same facts on demand.

View repository MCP docs

Hybrid Retrieval

Blend BM25 keyword hits with MiniLM embeddings. Dense + sparse vectors stay local for deterministic scoring.

Universal Loaders

Parse PDF (w/ OCR), Markdown, DOCX, TXT, Pages, and ChatGPT exports. Citations carry file, page, and offsets.

Resilient Index

SQLite-VSS when present, pure TypeScript fallback otherwise. Every chunk persists with UUIDv5 IDs and cached embeddings.

Agent Ready

search_local, get_doc, reindex, watch, stats, and import_chatgpt_export ship as MCP tools with strict Zod schemas.

Quickstart

npm install → npm run models:download → npm run index -- ./docs → npm run dev. All computation runs on-device and stays offline.

ChatGPT Imports

Convert conversations.json exports into Markdown with npm run chatgpt:to-md, then index alongside your dossiers.
